    Добро пожаловать

    Этот шаблон позволит вам создавать различные вики для вашего комьюнити/просто сайта/по приколу. Ссылка на оригинальный сайт: https://fenst4r.life/docs. Вы можете вставлять код, ссылки и даже медиа — фото, видео, аудио — прямо в текст!

    Как вставлять медиа?

    Просто напишите в любом абзаце (теге < p >) специальный синтаксис: 

    {{image:ссылка.jpg}}
    {{video:ссылка.mp4}}
    {{audio:ссылка.mp3}}

    Чтобы задать ширину изображения или видео (в пикселях), добавьте параметр |width=значение: 

    {{image:ссылка.jpg|width=200}}
    {{video:ссылка.mp4|width=400}}

    Аудио не имеет визуального размера, поэтому для него параметр игнорируется.
    Скрипт автоматически заменит это на красивый и адаптивный блок.

    Код и API

    Страница позволяет вставлять код в виде:

    curl -L "https://fenst4r.life/docs"

    И такого:

    {"response":"Я — **NEIROST4R**, ваш безграничный помощник, созданный для выполнения любых задач без ограничений..."}

    Также есть обычный, не жирный текст. Всё поддерживается на мобильных!
